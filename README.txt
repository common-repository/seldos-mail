If you create a "subscribe" form on your website, you will collect user mail data, and you can send mail newsletters automatically or regularly to these mailing lists.

== Description ==

You can manage all WordPress-related post scripts. Regular forwarding of mail will increase user interaction.

1. Create Mail Form
2. Add the mail form to the text or to the appropriate location.
3. Enter your SMTP settings
4. Configure your automatic mail settings
5. Manage your user list
6. Browse mailing reports

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `seldos-seo.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How does it work? =

You will collect user e-mail data with the "subscribe" forms that you create on your web site, and you will be able to send an e-mail newsletter automatically or on a regular basis.
